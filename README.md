# MarvelCharacters

It is a project for getting characters from Marvel service.

**Architectural Design Pattern:** MVVM with Architecture Components

**Reactive:** RxJava, RxAndroid

**IDE:** Android Studio

**SDK** Versions: Min Sdk 19, Compile Sdk 30, Target Sdk 30

**Programming Language:** Kotlin

**Dependency management:** Gradle

**Networking:** Retrofit

**Image Caching:** Picasso
